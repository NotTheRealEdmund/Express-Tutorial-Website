User Input Validation with JOI Validating Nested Object and Arrays

To make it easier to undertstand, we'll take out all the express stuff and just simulate a user's request for example.

To run the code, in VS code cmd, type in: node validate

JOI should work as expected for both valid and invalid data.

For valid data, you should see everything in personalInfo.

For invalid data, you'll see 'ValidationError' and a message containing the details such as '"streetAddress" is not allowed to be empty' if you changed the streetAddress to an empty string, because joi declared it to be required.
