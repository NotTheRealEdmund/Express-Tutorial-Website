Getting Started With EJS Templates With Express

EJS is a template language which allows us to write more dynamic web pages.

First we have to install EJS, in VS code cmd, type in: npm install ejs

You should see changes to package.json and package-lock.json files and node_modules folder.

Create a new file "template.js" inside the programs folder.

Create a new folder called "views" inside the programs folder.

Then create a new file "mytemplate.ejs" inside the views folder.

In "mytemplate.ejs", the "exampleVar" enclosed within the tags "<%=" and "%>" can be changed dynamically, based on the javascript file.

To run the server, in VS code cmd, type in: node template

To close server, press Ctrl-C

Go to browser and type: "localhost:3000/books".

You'll see that the webpage will show the same template in mytemplate.ejs but "exampleVar" is replaced with "books".

Search results get dynamically added to webpage.



Now we'll show multiple items in an array using for loop in EJS.

Create a new file "template2.js" inside the programs folder.

Then create a new file "mytemplate2.ejs" inside the views folder.

To run the server, in VS code cmd, type in: node template2

To close server, press Ctrl-C

Go to browser and type: "localhost:3000" and you'll see the list.

With EJS, that list will change dynamically, if the array in javascript changes, so will it in EJS.



Now we'll use if-else statements in EJS.

Create a new file "template3.js" inside the programs folder.

Then create a new file "mytemplate3.ejs" inside the views folder.

To run the server, in VS code cmd, type in: node template3

To close server, press Ctrl-C

Go to browser and type: "localhost:3000" and you'll see the list.

As per EJS, if-else statements also have dynamic properties.

In other words, if the age of a person is changed in javascript, the output in EJS might change.
