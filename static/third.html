Serving Static Files with Express

The static files can be html files, css files, javascript files, images, videos, client-side stuff etc.

Create a "demo.js" file inside the programs folder.

Create a "static" folder inside the programs folder and inside the newly created static folder create an "index.html" file.

To run the server, in VS code cmd, type in: node demo

To close server, press Ctrl-C

Go to browser and type: "localhost:3000" and you'll see the index.html file open.
